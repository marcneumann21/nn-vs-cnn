# Standard Neural Network vs. Convolutional Neural Network

This project compares a standard neural network implemented with NumPy to a Convolutional Neural Network (CNN) implemented with PyTorch for handwritten number recognition on custom data.

The goal is to evaluate which model performs better in terms of accuracy, training, and implementation complexity.

| Standard Neural Network (NumPy) | CNN (PyTorch) |
|---|---|
| Implemented from scratch | Uses PyTorch framework |
| (3) Fully connected layers | Convolutional layers |
| Manual backpropagation | Automatic differentiation |
| Simpler architecture | Optimized for image recognition |
